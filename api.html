API Documentation

GET

/api/listfiles
Outputs the names of the files directly inside of /projects, seperated by line breaks(\n)
/api/readfile
file
: Name of the file in /projects to read
Outputs the content of the file denoted by the file arguement

POST

/api/savefile
file
: Name of the file in /projects to read
content
: The data to write to the file
Outputs the content of the file denoted by the file arguement
